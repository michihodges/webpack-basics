# Express
Install the express, cors and body-parser packages:
```
npm i --legacy-peer-deps express@4.17.1
npm i --legacy-peer-deps cors@2.8.5
npm i --legacy-peer-deps body-parser@1.19.0
```

Create a `server.js` file:
```
touch server.js
```
