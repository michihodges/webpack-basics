## 1 Node and NPM
Open your CLI.</br>
Check Node JS is installed:
```
node --version
// or
node -v
```
Check NPM is installed:
```
npm --version
// or
npm -v
```
To install Node JS download [here](https://nodejs.org/en/).</br>
To install NPM:
```
npm install
```
